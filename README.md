# 🌍 Trip Advisor Clone

Trip Advisor Clone is a responsive travel website built using HTML and CSS.  
This project replicates the basic layout and design structure of the TripAdvisor website for learning and practice purposes.

## 📌 What I Built

This project includes:

- ✅ Responsive Navigation Bar  
- ✅ Font Awesome Icons Integration  
- ✅ Google Fonts (Poppins)  
- ✅ Animated Offer Banner  
- ✅ Destination Cards with Hover Effects  
- ✅ Reviews Section  
- ✅ Contact Page with Form  
- ✅ Sticky Navbar  
- ✅ Responsive Design (Mobile Friendly)
